## 📌 Project Description

The **Currency Exchange Calculator** is a Python program that takes an amount from the user and converts it into another currency based on predefined exchange rates.  

It was created to practice:

- Python input/output  
- Conditional statements  
- Basic arithmetic operations  
- Collaborative coding  

---

## ⚙️ Features

- 💰 Convert currency values  
- 🔢 Accept user input for amount  
- 🔁 Multiple currency options  
- ⚡ Instant calculation  
- 🐍 Built entirely using Python  

---

## 💻 Usage Example
Enter amount: 100
From currency (USD): USD
To currency (EUR): EUR
Converted Amount: 92.5 EUR
